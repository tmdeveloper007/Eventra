# Accessible Security Feedback

## Overview

Eventra provides accessible security notifications to ensure that authentication and authorization events are communicated effectively to all users, including users relying on assistive technologies.

## Security Announcements

The application announces:

- Authentication-required messages
- Session-expiration notifications
- Unauthorized-access feedback
- Offline synchronization security notices

## Accessibility Features

- ARIA live regions
- Assertive announcements for errors
- Status announcements for informational messages
- Screen-reader-compatible notification patterns
- Keyboard-accessible notification controls

## Examples

Authentication required. Please log in to register for events.

Security notice: Your session has expired. Please log in again to continue securely.

Security notice: Offline actions are pending, but your session has expired. Please log in again to synchronize them.

## Benefits

- Improved WCAG compliance
- Better experience for screen-reader users
- Consistent security communication
- More inclusive authentication workflows
