# Environment Security Dashboard

## Overview

Eventra includes a development-only Environment Security Dashboard that validates environment configuration and highlights potential security issues.

## Features

- Required environment variable validation
- Missing configuration detection
- Localhost detection in production configurations
- Security recommendations
- Visual status indicators

## Security Benefits

- Reduces configuration drift
- Prevents insecure deployments
- Improves visibility into environment configuration
- Encourages secure development practices

## Production

The dashboard is available only during development and is excluded from production builds.
