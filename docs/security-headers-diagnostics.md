# Security Headers Diagnostics Panel

## Overview

Eventra includes a development-only Security Headers Diagnostics Panel that provides visibility into security header configuration and CSP reporting.

## Features

- Content Security Policy inspection
- CSP reporting endpoint detection
- Security recommendations
- Status indicators
- Development-only diagnostics

## Benefits

- Faster debugging of security configuration
- Easier validation of CSP deployment
- Improved visibility into security posture
- Better developer experience

## Production

The diagnostics panel is excluded from production builds.
