# Contributing to Eventra

Thank you for contributing to Eventra! 🎉

## How to Contribute

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Submit a Pull Request

## GSSoC'26 Contribution Points

### Label-Based Scoring Metrics
- Level 1 GSSoC Pull Request → **3 Points**
- Level 2 GSSoC Pull Request → **7 Points**
- Level 3 GSSoC Pull Request → **10 Points**

### Milestone Boosters
- +5 Bonus for 5+ merged PRs
- +10 Bonus for 10+ merged PRs

### Elite Achievement Badges
- 🥇 Grandmaster — Rank #1 Overall
- ⭐ Champion — Rank #2 Overall
- 🏅 Elite — Rank #3 Overall
- 🔥 PR Machine — Merged 10+ PRs
