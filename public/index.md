# Eventra - Platform for Builders

Eventra is a modern platform designed for builders to discover hackathons, showcase projects, and connect with contributors.

## Key Features

- **Discover Hackathons**: Stay updated with upcoming and ongoing hackathons.
- **Showcase Projects**: Submit your projects and get feedback from the community.
- **Connect with Contributors**: Find collaborators for your next big idea.
- **Leaderboards**: Compete and see where you stand among top builders.

## API Access

We provide RESTful APIs for automated interaction with our platform.

- **API Catalog**: /.well-known/api-catalog
- **API Documentation**: /apiDocs

## Agent Support

Eventra is built to be AI-agent friendly. Use the `Accept: text/markdown` header to receive a markdown version of our pages.
